# 🌆 AI for Better Living & Smarter Communities

An AI-powered Decision Intelligence Platform that transforms how communities manage resources, engage citizens, and make data-driven decisions./

## 🚀 Features

- **Dashboard** - Real-time community metrics and AI predictions
- **Community** - Events, services, and engagement tracking
- **Safety** - Incident monitoring and emergency resources
- **Healthcare** - Facilities availability and statistics
- **AI Assistant** - Chatbot for instant community information

## 🛠️ Tech Stack

- **Frontend**: React.js
- **Backend**: Express.js
- **AI/ML**: Simulated intelligence for insights
- **Styling**: Modern CSS with gradients

## 📦 Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The app will be available at:
- Frontend: http://localhost:3000
- Backend: http://localhost:5000
